# Clone do Spotify 🎧

Este é um projeto que replica a interface de usuário do Spotify, desenvolvido para praticar habilidades avançadas de Front-End, design responsivo e organização de componentes.

## 🚀 Tecnologias Utilizadas

* *HTML5:* Estrutura semântica.
* *CSS3:* Estilo com Flexbox e Layout de Grade.
* *JavaScript:* Lógica de interação e manipulação do DOM.
* *Git/GitHub:* Controle de versão e implantação.

## 🛠 ️ Funcionalidades

- [x] Interface totalmente responsiva (Mobile-First).
- [x] Menu lateral com efeitos de navegação.
- [x] Lista de reprodução dinâmica.
- [x] Reprodutor de música visual na parte inferior.
- [x] Efeitos avançados do Hover para uma melhor experiência do usuário (UX).

## 💡 Aprendizados

Durante o desenvolvimento deste projeto, fortaleci meu conhecimento em:
1. *Layouts complexos:* Usando CSS Grid para organizar a biblioteca e o conteúdo principal.
2. *Interatividade:* Uso de JavaScript para lidar com eventos do usuário.
3. *Otimização:* Estrutura de código limpa e fácil de manter.
